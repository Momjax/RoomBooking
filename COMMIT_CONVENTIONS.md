# Conventions de Commit — RoomBooking

## Format
```
<type>: <description courte>
```

## Types
| Type | Usage |
|------|-------|
| `feat` | Nouvelle fonctionnalité |
| `fix` | Correction de bug |
| `docs` | Documentation |
| `style` | CSS, formatage (pas de changement de logique) |
| `refactor` | Refactoring de code |
| `test` | Ajout ou modification de tests |
| `chore` | Config, dépendances, maintenance |

## Exemples
```
feat: Jalon 1 & 2 - Init Symfony + Config MySQL
feat: Jalon 3 - Entités Doctrine + Repositories
chore: Jalon 4 - Config Git + conventions de commit
feat: Jalon 5 - Migrations + Fixtures
feat: Jalon 6 - Auth login/logout
```
